# 专利附图绘制说明

以下为专利"一种基于双阶段协同评估的配电网储能优化配置方法"中 6 张附图的详细绘制说明。

---

### 图1: 整体技术路线图

**图类型**：流程图（自上而下分层结构）

**表达目的**：展示从数据加载到最优方案输出的完整处理流程，使读者在一张图中获得方法全局概览。

**整体布局**：
- 方向：自上而下，分阶段用水平色带分隔
- 画布比例：A4 竖版，宽度约 18 cm，高度约 24 cm

**元素清单**：

| 编号 | 元素名称 | 类型 | 颜色建议 | 文字内容 |
|------|---------|------|---------|---------|
| A1 | 输入数据 | 圆柱体 | 浅蓝 #D6EAF8 | 负荷模型 / 光伏模型 / 分时电价 / OpenDSS模型 / 储能策略 |
| B1 | 种群初始化 | 矩形 | 浅灰 #E5E7E9 | 随机生成 S₀ 个 [P_rated, T_duration] |
| C1 | 典型日快速评估 | 矩形 | 浅橙 #FDEBD0 | 代表日聚类 → 日前调度 → 滚动执行与配电网约束校核 |
| C2 | 经济性筛选 | 菱形 | 浅橙 | 回收期≤阈值 ∧ NPV/投资≥阈值 ∧ 现金流转正？ |
| C3 | 全年精确评估 | 矩形 | 深橙 #E67E22 | 是 → 365天完整仿真 |
| C4 | 保留快评结果 | 矩形 | 浅灰 | 否 → 保留典型日评估结果 |
| D1 | 约束分级比较 | 矩形 | 浅绿 #D5F5E3 | 电网安全 → 设备可行 → 经济可行 → NPV |
| D2 | 自适应种群调节 | 矩形 | 浅绿 #D5F5E3 | NPV改进率 → 扩大/收缩/保持 |
| D3 | 下一代生成 | 矩形 | 浅绿 #D5F5E3 | 精英保留+锦标赛选择+交叉变异 |

- A1 → B1 → C1 → C2
- C2 → 是 → C3 → D1

- D4 → 是 → E1

**关键标注**：
- 左侧竖排标注各阶段名称："一、初始化""二、双阶段评估""三、进化搜索""四、输出"
- C1-C4 处用虚线框标注"核心创新：双阶段协同评估"
- D1-D2 处用虚线框标注"GA 改进"

### 图3: 自适应种群规模调节示意图

**图类型**：分区示意图

**表达目的**：展示 NPV 改进率与种群规模调节决策之间的映射关系。

**整体布局**：
- 方向：上部为决策流程图，下部为数值轴分区示意
- 画布比例：横版 16:9

**元素清单**：

| 编号 | 元素名称 | 类型 | 颜色建议 | 文字内容 |
|------|---------|------|---------|---------|
| D1 | 计算改进率 | 矩形 | 浅灰 | δ = \|NPV_g - NPV_{g-1}\| / \|NPV_{g-1}\| |
| D2 | 增长判断 | 菱形 | 浅绿 | δ > θ_growth？ |
| D3 | 扩大种群 | 矩形 | 深绿 | 是 → S_{g+1} = min(S_g+2, S_max) |
| D4 | 收缩判断 | 菱形 | 浅红 | δ < θ_shrink？ |
| D5 | 收缩种群 | 矩形 | 深红 | 是 → S_{g+1} = max(S_g-2, S_min) |
| D6 | 保持种群 | 矩形 | 浅灰 | 否 → S_{g+1} = S_g |
| R1 | 改进率数轴 | 水平数轴 | 黑色 | 标注 0, θ_shrink, θ_growth 在数轴上的位置 |
| R2 | 分区标注 | 文本标注 | 对应颜色 | 收缩区 [0, θ_shrink) / 保持区 [θ_shrink, θ_growth] / 增长区 (θ_growth, ∞) |

**连接关系**：
- D1 → D2
- D2 → 是 → D3
- D2 → 否 → D4
- D4 → 是 → D5
- D4 → 否 → D6

**关键标注**：
- 增长区标注"探索：改进显著，扩大搜索范围"
- 保持区标注"平稳：改进适中，维持当前规模"
- 收缩区标注"精化：趋于停滞，收缩以聚焦局部"
- 标注典型参数值：θ_growth = 0.15, θ_shrink = 0.05

### 图4: 约束分级比较方法示意图

**图类型**：层级金字塔图

**表达目的**：以直观的层级结构展示三级约束的分类、物理含义和逐级比较规则，以及与罚函数法的对比。

**整体布局**：
- 方向：自上而下的三级金字塔 + 右侧对比框
- 画布比例：横版 16:9

**元素清单**：

| 编号 | 元素名称 | 类型 | 颜色建议 | 文字内容 |
|------|---------|------|---------|---------|
| L1 | 第一级框 | 矩形（最宽） | 深红 #C0392B | 电网安全约束：电压越限(pu) / 线路过载(%) / 变压器越限(h+kw) |
| L2 | 第二级框 | 矩形（中等宽） | 橙色 #E67E22 | 设备可行约束：储能时长越限(h) / 循环次数超限 |
| L3 | 第三级框 | 矩形（最窄） | 黄色 #F1C40F | 经济可行约束：现金流为负 / 回收期超限 |
| L4 | 目标层 | 矩形（最窄） | 绿色 #2ECC71 | 经济目标：-NPV / 回收期 / 投资额 |
| R1 | 比较规则 | 箭头+文本 | 黑色 | 从第一级开始逐级比较；首次出现差异的级别决定排序；同级全相等才比较下一级 |
| R2 | 罚函数对比框 | 虚线框 | 浅灰 | "vs 罚函数法：fitness = NPV - w₁·V_safety - w₂·V_device - w₃·V_economy" |
| R3 | 罚函数缺陷标注 | 文本 | 红色 | "高 NPV 可'补偿'任何约束违反——安全优先性无法保证" |

**关键标注**：
- 左侧标注"优先级高→低"，箭头自上而下
- 各级框宽度递减，视觉化优先级的逐级降低
- R2 对比框底部标注"本方法的词典序比较不允许多目标之间的补偿——电网安全优先性不可被经济指标稀释"

### 图5: 代表日聚类签名空间示意图

**图类型**：三维散点图

**表达目的**：直观展示全年 365 天在负荷均值-负荷标准差-电价均值三维签名空间中的分布及分组效果。

**整体布局**：
- 方向：三维坐标系
- 画布比例：横版 16:9

**元素清单**：

| 编号 | 元素名称 | 类型 | 颜色建议 | 文字内容 |
|------|---------|------|---------|---------|
| S1 | X轴 | 坐标轴 | 黑色 | 负荷均值签名：bin = floor(mean_load / 50) |
| S2 | Y轴 | 坐标轴 | 黑色 | 负荷标准差签名：bin = floor(std_load / 30) |
| S3 | Z轴 | 坐标轴 | 黑色 | 电价均值签名：bin = floor(mean_tariff × 20) |
| S4 | 数据散点 | 三维散点 | 同组同色 | 全年 365 天的散点分布 |
| S5 | 代表日标记 | 大号散点 | 红色边框 | 每组签名空间中的代表日（各组的首日） |
| S6 | 分组边界 | 虚线网格 | 灰色 | 离散化步长划分的立方体网格 |

**关键标注**：
- 不同签名组用不同颜色区分，标注总组数（如"共 14 组"）
- 坐标轴下方标注物理映射：
  - "50kW/bin → 负荷量级（套利机会的绝对值）"
  - "30kW/bin → 日内波动性（储能调节空间）"
  - "0.05元/kWh per bin → 电价水平（充放电价差信号）"
- 底部标注："三维签名直接编码储能套利决策的核心变量，具有明确的物理可解释性"

### 图6: 日前调度多求解器链式调用流程图

**图类型**：流程图（含分支和回退）

**表达目的**：展示凸优化日前调度中依次尝试 OSQP、ECOS 和 SCS 三个求解器的逻辑链路，以及全部失败后回退到规则调度的完整路径。

**整体布局**：
- 方向：从左到右的横向链式结构
- 画布比例：横版 16:9

**元素清单**：

| 编号 | 元素名称 | 类型 | 颜色建议 | 文字内容 |
|------|---------|------|---------|---------|
| P1 | 输入 | 圆角矩形 | 浅蓝 | 某代表日 24h 负荷/光伏/电价/SOC初值 |
| P2 | 构建凸优化问题 | 矩形 | 浅灰 | cvxpy 建模：变量+目标+约束 |
| S1 | 尝试OSQP | 矩形 | 浅绿 | OSQP: max_iter=50000, eps_abs=1e-3 |
| S1J | OSQP结果判断 | 菱形 | 浅绿 | 求解成功？ |
| S2 | 尝试ECOS | 矩形 | 浅橙 | ECOS: max_iters=8000, abstol=1e-7 |
| S2J | ECOS结果判断 | 菱形 | 浅橙 | 求解成功？ |
| S3 | 尝试SCS | 矩形 | 浅紫 | SCS: max_iters=50000, eps=5e-6 |
| S3J | SCS结果判断 | 菱形 | 浅紫 | 求解成功？ |
| F1 | 规则回退调度 | 矩形 | 浅红 | 电价≤25%分位→充电 / ≥75%分位→放电 |
| R1 | 多解择优 | 矩形 | 浅灰 | 按求解状态+松弛量+末端SOC误差+目标值排序 |
| O1 | 输出调度计划 | 圆角矩形 | 浅蓝 | 24h 充放电功率+SOC轨迹 |

**连接关系**：
- P1 → P2 → S1 → S1J
- S1J → 是 → R1
- S1J → 否 → S2 → S2J
- S2J → 是 → R1
- S2J → 否 → S3 → S3J
- S3J → 是 → R1
- S3J → 否 → F1
- F1 → R1
- R1 → O1

**关键标注**：
- 三个求解器上下排列，用箭头连成链，标注"依次尝试"
- 求解成功路径用绿色箭头，失败路径用红色箭头
- 规则回退框旁标注："仅在全部求解器失败时触发"

**特殊说明**：
- 若有多个求解器返回可行解（如 OSQP 返回近似最优，ECOS 返回精确最优），R1 择优步骤会选出质量最高的解——并非第一个成功的求解器就一定被采用
