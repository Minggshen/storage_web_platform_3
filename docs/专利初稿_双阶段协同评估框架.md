# 专利技术交底书

## 一、专利名称

一种基于双阶段协同评估的配电网储能优化配置方法

## 二、发明人/申请人/地址

（留空）

## 三、本专利的应用领域

本发明涉及配电网储能规划与优化配置领域，尤其涉及一种面向工商业配电网节点的、基于典型日快速评估与全年精确评估双阶段协同的储能功率与容量优化配置方法，可在保证配电网电压与线路安全约束的前提下实现储能全生命周期经济性的最优决策。

## 四、本专利的任务/要解决的技术问题

在工商业配电网中为给定节点配置电池储能系统，需要同时确定两个核心参数——额定功率（kW）和额定容量（kWh）。这两个参数的优化需要在经济性目标（全生命周期净现值最大化、投资回收期最小化）和安全性目标（储能接入后不导致配电网电压越限、线路过载和变压器过载）之间寻求平衡。

该问题的计算瓶颈在于每一次候选方案的评估都需要执行完整的年度时序调度仿真和配电网潮流校核，以获取该方案的全生命周期财务指标和安全指标。一个完整的年度仿真涉及 365 天×24 小时的逐时日前调度优化、滚动实时执行校验、以及配电网潮流计算。使用工业级配电网仿真工具（如 EPRI OpenDSS）完成一次完整的年度校核通常需要数分钟，而进化优化需要评估数百个候选方案——两者的组合在计算上构成了根本性的矛盾。

现有方法的典型应对策略存在以下不足：（一）使用简化的线性约束替代真实配电网潮流校核，丧失了配电网安全验证的物理真实性；（二）对所有候选方案采用同一评估精度，计算资源在大量不可行或经济性差的方案上被均匀消耗，计算效率低下；（三）在约束处理上普遍采用罚函数法，不同约束的优先级差异无法体现且惩罚系数依赖人工调参；（四）采用固定种群规模，不随优化进展动态调整。

本专利的任务是：（1）提出一种双阶段协同评估方法，先以典型日压缩全年时序对全部候选方案进行快速评估，再对通过经济性筛选的少数有潜力方案执行完整的全年精确评估，从而在搜索广度和物理仿真精度之间建立可自动调节的平衡机制；（2）提出一种约束分级比较方法，将约束按物理含义分为不同等级并逐级比较，替代罚函数加权，从根本上消除惩罚系数调参的困难；（3）提出一种优化目标改进率驱动的自适应种群规模调节策略，使种群规模随搜索进展动态调整。

## 五、已有技术/产品的不足

在现有配电网储能优化配置相关技术中，基于遗传算法的多目标优化方法已被多件专利采用。如专利 CN202210862513.8（国网湖北省电力有限公司经济技术研究院，2022）提出了一种基于改进遗传算法的电力系统多目标储能优化方法，在电源侧、电网侧和负荷侧均配置储能并构建以总成本最小为目标的多目标优化模型，其不足之处在于未将配电网的真实潮流约束纳入候选方案评估过程，优化结果的配电网安全性无法得到物理验证，且对所有候选方案采用同一评估精度，计算资源的分配未考虑候选方案之间的质量差异。专利 CN119171530A（国网福建省电力有限公司经济技术研究院，2024）提出了一种配电网多目标联合规划方法，采用改进第三代非支配排序遗传算法结合逻辑混沌映射初始化和置信区间上界局部搜索进行分布式光伏与储能的联合规划，该方法虽嵌套牛顿-拉夫逊潮流计算进行约束校核，但仍对所有候选方案执行同一精度的评估，未形成按候选方案经济可行性进行差异化评估的分层策略，大量计算资源在不可行或经济性差的方案上被均匀消耗。在约束处理方面，现有方法普遍依赖罚函数法，如专利"一种储能系统选址定容方法"采用模糊隶属度函数进行最终方案选择，但对约束违反的处理仍采用传统的罚函数加权方式，惩罚系数的设置依赖人工经验，且无法体现电网安全约束相对于经济指标之间的绝对优先关系。在种群管理方面，现有方法均采用固定种群规模，不随优化进展动态调节——在搜索初期固定的小种群可能遗漏优质解区域，在搜索后期固定的大种群存在计算资源浪费。因此，有必要提出一种在评估精度差异性、约束处理层次性和种群规模自适应性三个维度上进行系统性改进的配电网储能优化配置方法。

## 六、本专利的内容

### 6.1 总体方案

本发明提出一种基于双阶段协同评估的配电网储能优化配置方法，其核心算法命名为双阶段协同遗传算法（Two-Stage Co-Evolutionary Genetic Algorithm, TSC-GA）。如图1所示，该方法的整体技术路线如下。

加载给定配电网节点的年度负荷模型、光伏出力模型、分时电价序列、配电网 OpenDSS 电路模型和储能设备技术经济模型（策略）。进化优化器在策略给定的搜索空间内生成候选配置方案的初始种群，每个候选方案由额定功率和额定容量两个连续决策变量组成。此后进入逐代迭代：

**第一步——典型日快速评估**：对种群中通过快速筛选的每个候选方案，采用代表日聚类方法将全年 365 天压缩为若干个代表日组，对每组代表日执行日前凸优化调度、滚动执行和 OpenDSS 配电网约束校核，获得该方案的近似经济指标（NPV、回收期、年度现金流）。约束违反指标（电压越限、线路过载、变压器越限等）作为配电网潮流校核的自然副产品同时获得。

**第二步——经济性筛选**：对每个候选方案进行三重经济门槛判断。通过门槛的方案进入全年精确评估；未通过的方案评估至此结束，保留典型日快速评估结果作为最终评估值。

**第三步——全年精确评估**：仅对通过经济性筛选的候选方案执行，对全年 365 天均进行完整的日前调度、滚动执行和 OpenDSS 配电网约束校核，并以精确结果覆盖快评结果。

两个评估阶段使用同一配电网仿真引擎，仅在时间采样密度上存在差异（代表日 vs 全年逐日），确保快速评估与精确评估之间的信号一致性，消除传统代理模型中代理模型与真实评估之间的语义偏差。

**第四步——约束分级比较与种群迭代**：当代全部候选方案的评估完成后，采用约束分级比较方法——以词典序逐级比较电网安全约束、设备可行约束和经济可行约束——对种群进行排序。排序结果用于精英保留和锦标赛选择。随后根据优化目标改进率自适应调节下一代种群规模，并通过交叉变异生成新一代候选方案。

迭代至预设代数后，从最终的 Pareto 存档中以安全-经济权衡系数 s 选出该策略下的最优折中方案并输出。若该方案未经过全年精确评估，则补做一次全年精确评估后重新比选。对于存在多个候选策略的场景，对各策略分别独立执行上述 TSC-GA 优化流程，各策略最优方案汇总后进行跨策略比选，输出全局推荐配置方案。

以下对各技术模块进行详细说明。

### 6.2 决策变量与搜索空间定义

对于给定的储能设备技术经济模型——定义了充放电效率、单位能量成本、单位功率成本、SOC 运行范围、最大充放电倍率、循环寿命、年度循环次数上限、退化成本系数等模型参数——进化优化器在该模型给定的框架内搜索最优的连续决策变量：

```
x = [P_rated, T_duration]                                   (1)
```

式中 P_rated 为储能系统的额定功率（kW），T_duration 为额定储能时长（h）。额定容量由 E_rated = P_rated × T_duration 确定（kWh）。搜索可行域由策略给定的功率下界 P_min 和上界 P_max、储能时长下界 T_min 和上界 T_max 共同约束。T_min 和 T_max 由策略的额定能量与额定功率之比（额定储能时长）及该策略允许的储能时长偏差范围共同确定，反映同一产品型号在电池模组配置数量上的可调节范围。

### 6.3 优化目标函数

在由 §6.2 定义的搜索空间内，进化优化器以最小化方向优化经济维度的三个目标，以最大化方向优化安全维度的一个目标：

```
min  f_econ(x) = [ f₁(x), f₂(x), f₃(x) ]                 (2)

f₁(x) = -NPV(x)        // 净现值负值，最小化等价于最大化 NPV
f₂(x) = PBP(x)         // 简单投资回收期（年），最小化
f₃(x) = CAPEX(x)       // 储能系统初始投资额（元），最小化

max  f_safety(x) = SAFETY(x)                              (3)

SAFETY(x) = SAFETY_grid(x) + SAFETY_dev                    (3)

SAFETY_grid(x) = w_imp × ΔS̃(x) - [ w_tr_h × H̃_tr(x) + w_tr_s × P̃_tr(x)
               + w_volt × H̃_volt(x) + w_line × H̃_line(x) ]

SAFETY_dev = α × L_safety + β × L_cooling + γ × (1 - R_soc)
           + δ × (1 - R_c_rate)
```

式中各符号含义如下：

**量纲归一化处理**：由于各分量的物理单位和数值范围不同，在加权求和之前先进行归一化处理，将所有分量映射至 [0, 1] 或 [-1, 1] 区间。H̃_tr = H_tr / 8760（以全年小时数为自然上限），P̃_tr = P_tr / P_trans（以变压器额定容量 kW 为自然上限），H̃_volt = H_volt / 8760，H̃_line = H_line / 8760，ΔS̃ = ΔS / 8760。经归一化后，所有权重 w 仅反映各分量在安全评价中的相对重要性，不再承担量纲转换的职责。

**安全改善量**：ΔS 为储能接入后配电网安全越限总小时数相对于未接入储能的基准场景的减少量（h），即 ΔS = S_baseline - S_storage。ΔS 为正表示储能接入改善了配电网安全，为负表示储能接入恶化了配电网安全。

**安全越限指标**（全部来源于配电网 OpenDSS 潮流计算和年度时序仿真）：H_tr 为变压器有功功率全年越限总小时数；P_tr 为变压器全年最大越限有功功率（kW）；H_volt 为全网母线电压全年越限总小时数；H_line 为线路电流全年过载总小时数。上述四个指标共同反映储能接入后对配电网节点电压、线路负载和变压器负载三个维度的实际物理影响。权重 w_imp、w_tr_h、w_tr_s、w_volt、w_line 反映了不同越限类型对配电网安全的相对危害程度。

**设备安全分量**：SAFETY_dev 为由储能策略（设备技术经济模型）给定的设备固有安全评价值，不随候选方案的功率和容量变化。式中 L_safety 为归一化安全等级（高=1.0，中=0.5，低=0.0），L_cooling 为归一化冷却方式能力（液冷=1.0，风冷=0.5），R_soc 为 SOC 运行区间宽度（soc_max - soc_min）与额定区间 0.8 之比，R_c_rate 为额定充放电倍率与行业基准倍率 1C 之比。α、β、γ、δ 为各子项的权重系数。SAFETY_grid 与 SAFETY_dev 经归一化后均映射至相近的数值范围内，两者在总安全指标 SAFETY 中等权相加，分别代表配电网侧和设备侧对储能系统综合安全性的贡献。

SAFETY > 0 表示储能接入对配电网安全具有净正向改善；SAFETY < 0 表示储能接入对配电网安全存在净恶化风险。优化方向为最大化 SAFETY。

上述经济目标与安全目标之间存在冲突关系：容量更大的储能系统通常可获得更高的 NPV（套利收益更多），但初始投资也更高；安全目标可能与经济目标同向——在负荷高峰期放电支撑电压可在提升经济性的同时改善安全性——但也可能反向——若充放电策略过度追求峰谷套利，储能本身也可能引发新的电压越限或线路过载。因此，本方法在进化搜索过程中维护一个 Pareto 非支配存档，保留在经济性和安全性之间取得不同权衡的 Pareto 最优折中集合。

进化优化完成后，通过一个安全-经济权衡系数 s ∈ [0, 1] 从 Pareto 存档中选出最终的最优折中方案。s = 0 表示纯经济偏好（仅关注 NPV、回收期和投资额），s = 1 表示纯安全偏好（仅关注 SAFETY），s = 0.5 表示经济与安全并重。该系数可由用户根据实际工程需求调节。

### 6.4 双阶段协同评估框架

本模块是本发明的核心创新。其基本思路是：对全部候选方案先以较低的计算代价进行快速评估，再对少数经济上有潜力的方案进行精确评估——两个阶段使用同一配电网仿真引擎，仅在时间采样密度上存在差异，确保快速评估与精确评估之间的信号一致性，消除传统代理模型中代理与真实评估之间的语义偏差。如图2所示。上述四个目标函数的计算均遵循下述双阶段评估流程——典型日快速评估给出近似的目标值，全年精确评估给出最终的目标值。

#### 6.4.1 第一阶段：典型日快速评估

典型日快速评估的目标是以远低于全年完整仿真的计算代价，获取候选方案的近似经济指标。由于评估过程已包含 OpenDSS 配电网潮流校核，约束违反指标（电压越限、线路过载、变压器越限等）作为校核的自然副产品同时获得——无需为该步骤单独执行约束分析。

首先，对全年 365 天进行代表日聚类。不同于传统的基于负荷曲线形状的 k-means 聚类，本方法采用多维签名自适应分组策略——以每日负荷均值、负荷标准差和电价均值的离散化签名对全年各天进行分组，具有相同签名的天归入同一代表日组，每组仅选取一个代表日。由此将全年压缩为若干（通常十余个）代表日组。组间递推储能 SOC 状态，保持长期运行的 SOC 连续性。

对每个代表日，构建以该日 24 小时为决策时域的凸优化日前调度模型。决策变量包括各小时的充电功率、放电功率和辅助服务预留功率。目标函数为最大化单日运营收益：

```
max Σ_h [ tariff[h] × (p_dis[h] - p_ch[h])
        + cap_price[h] × p_srv[h] + del_price[h] × α[h] × p_srv[h]
        - d_deg × (p_ch[h] + p_dis[h] + w_srv × α[h] × p_srv[h])
        - w_tr × slack[h] ]                                   (4)
```

式中 tariff[h] 为分时电价（元/kWh），p_ch[h] 和 p_dis[h] 为充放电功率，p_srv[h] 为辅助服务功率，d_deg 为退化成本系数，slack[h] 为变压器容量松弛变量，w_tr 为变压器越限惩罚权重，α[h] 为辅助服务调用概率。

约束条件包括 SOC 递推方程、SOC 范围约束、充放电功率上限约束和变压器容量软约束。为增强求解鲁棒性，采用多求解器链式调用策略——依次尝试 OSQP、ECOS 和 SCS 三个凸优化求解器，按求解质量和松弛量等指标择优；若全部失败则回退到基于电价分位数的规则调度策略。

日前调度完成后，通过滚动执行将计划充放电功率与实时的配电网约束进行协调。对每个小时，将计划值输入配电网约束校验模块，获取该小时的最大允许充放电功率、变压器限值和电压越限信息；实际执行功率在计划值和约束上限之间取较小值。将全天的校验结果汇总为该代表日的调度评估数据，乘以该组的天数权重后累加到年度汇总中。

#### 6.4.2 经济性筛选

典型日快速评估完成后，对候选方案进行三重经济门槛判断：

```
触发全年精确评估 ⇔ (回收期 ≤ T_max)
                      ∧ (NPV / 初始投资 ≥ R_min)
                      ∧ (年度现金流 > 0)                     (5)
```

式中 T_max 为最大允许回收期，R_min 为 NPV 与初始投资之比的最低门槛。三个条件分别对应"项目能否在合理期限内回本""净现值相对于投入是否不成比例亏损""年度运营是否产生正向现金流"。

通过三重门槛的候选方案进入全年精确评估。未通过的方案评估至此结束——保留典型日快速评估的结果作为该方案的最终评估值，其约束违反指标作为快评的自然副产品一并保留，不触发全年精确评估。在典型场景中，该门槛可过滤掉大部分候选方案。

#### 6.4.3 第二阶段：全年精确评估

对于通过经济性筛选的候选方案，全年精确评估阶段对全年 365 天均执行完整的日前调度、滚动执行和配电网约束校核，获得 365×24 小时的完整配电网运行数据，包括每小时的储能充放电功率、SOC 轨迹、电网交换功率、变压器负载率、母线电压和线路电流分布。基于全年逐时数据的完整集合，计算全生命周期财务指标（NPV、IRR、回收期）和配电网安全指标（变压器越限总小时数、最大越限功率、电压越限小时数、线路过载小时数等）。

全部代数迭代完成后，从最终的 Pareto 存档中以安全-经济权衡系数 s 选出最优折中方案。若该方案在此前的进化过程中未触发全年精确评估（即仅经过典型日快速评估），则对其进行一次补评估——执行完整的全年精确评估并以精确结果更新该方案的评估值——然后重新比选。该补评估仅在进化结束后对最优折中方案执行一次，不参与进化迭代过程，目的是确保最终输出的推荐方案经过了完整的配电网安全验证。

### 6.5 约束分级比较方法

本方法取代传统罚函数法，采用约束分级比较机制对进化种群中的候选方案进行评价。其核心原则是：不同物理含义的约束具有不同的优先级——电网安全约束的违反相对于设备可行约束的违反具有绝对的优先排序权，设备可行约束相对于经济可行约束亦然，经济指标仅在所有约束等级均相同时才影响排序。

#### 6.5.1 约束的三级分类

每个候选方案经全年仿真评估后，提取其各项约束违反指标。这些指标按物理含义归入以下三个等级：

**第一级——电网安全约束**：包括母线电压越限量（pu）、线路过载百分比（%）、变压器全年越限小时数和变压器最大越限功率（kW）。该级约束反映储能接入后对配电网运行安全的实际影响，全部来源于配电网潮流计算，具有最高的优先级。

**第二级——设备可行约束**：包括储能时长偏离策略允许范围的绝对值（h）和年度等效满循环次数超出策略年度循环上限的差值。该级约束反映候选方案在物理上是否可被给定的储能设备执行——时长超限意味着电池无法在要求的持续时间内输出额定功率，循环超限意味着设备将以超出设计寿命的速度衰减。

**第三级——经济可行约束**：包括年度运营现金流的负值差额和投资回收期超出最大允许回收期的差额。该级约束反映项目在经济上是否具备可行性。

#### 6.5.2 级内加权与级间比较

由于同级内部的不同指标具有不同的物理单位和数值范围，对每级内的指标进行加权求和，得到一个无量纲的级内总违反标量。各级加权系数反映了指标在该级内的相对重要性。上述三级约束的总违反标量分别记为 V_safety、V_device 和 V_economy。

候选方案的比较采用词典序逐级比较——从第一级开始，在第一个出现差异的级别上，违反量较小者胜出：

```
比较规则：对于候选方案 A 和 B，
  若 V_safety(A) < V_safety(B)，则 A 优于 B；
  若 V_safety(A) > V_safety(B)，则 B 优于 A；
  若 V_safety(A) = V_safety(B)，则比较 V_device；
  若 V_device 也相等，则比较 V_economy；
  若 V_economy 也相等，则比较经济目标函数（-NPV）。           (6)
```

#### 6.5.3 与罚函数法的本质区别

罚函数法的比较方式为 `fitness = NPV - w_safety × V_safety - w_device × V_device - w_economy × V_economy`——所有约束违反被折算为经济代价后与 NPV 相加减。其根本缺陷在于：足够高的 NPV 可以"补偿"任何级别的约束违反——一个电压严重越限但经济效益极高的方案，在罚函数框架下可能排在一个电压完全合规但经济效益略低的方案之前。

本方法的分级比较规则从机制上避免了这种错误：电网安全违反量只要不为零，无论数值多小、无论 NPV 多高，永远排在电网安全无违反的方案之后。电网安全相对于经济指标具有不可被补偿的绝对优先权。

以上约束分级比较规则定义了进化搜索中个体优劣的评价准则。基于该准则，每一代均可确定当代的最优个体及其对应的净现值。下面引入基于该最优净现值代际改进率的自适应种群规模调节策略。

### 6.6 自适应种群规模调节

传统进化优化的种群规模在整个搜索过程中保持固定。本方法根据优化进展动态调整种群规模，如图3所示。

设第 g 代和上一代的最优净现值分别为 NPV_g 和 NPV_{g-1}，定义代际改进率：

```
δ_g = |NPV_g - NPV_{g-1}| / |NPV_{g-1}|                     (7)
```

种群规模 S_{g+1} 的调节规则为：

```
               ┌ min(S_g + 2, S_max),   if δ_g > θ_growth
S_{g+1} =      ┤ max(S_g - 2, S_min),   if δ_g < θ_shrink     (8)
               └ S_g,                   otherwise
```

式中 θ_growth 为增长阈值，θ_shrink 为收缩阈值，S_min 和 S_max 为种群规模的下界和上界。调节逻辑为：当净现值改进显著（δ_g > θ_growth）时，当前搜索方向富有成效，应扩大种群以增强探索能力；当净现值改进趋于停滞（δ_g < θ_shrink）时，当前区域已接近局部最优，应收缩种群以减少计算浪费并加强局部精化。

该调节策略利用的是优化目标本身的经济信号（净现值改进率）而非通用的种群多样性指标，调节逻辑简明且与优化任务直接对齐。

### 6.7 算法详细程度说明

上述各模块中，凸优化调用 cvxpy 库实现，日前调度问题依次尝试 OSQP、ECOS 和 SCS 三个求解器；进化优化的锦标赛规模为 3，精英保留比例为种群规模的 1/4，变异率为 0.35，重初始化比例为 0.20；代表日聚类的签名离散化步长为 50kW、30kW 和 0.05 元/kWh；经济性筛选的最大回收期默认为 10 年，NPV/投资最低门槛默认为 -0.20；自适应种群调节的增长阈值默认为 0.15，收缩阈值默认为 0.05，种群下界默认为 12，上界默认为 24。本领域技术人员根据上述公式、流程描述和参数设定，可在 Python 3.11 科学计算环境中完整复现本方法。

## 八、本专利的效果

（1）通过双阶段协同评估方法——典型日快速评估（代表日聚类→日前调度→滚动执行与约束校核）→ 经济性筛选（三重门槛）→ 全年精确评估（365 天完整仿真）——解决了进化优化搜索广度与配电网物理仿真精度之间的根本矛盾。快速评估阶段将全年仿真压缩至若干代表日，使全部候选方案以低成本获得在配电网物理约束下经过校准的评估信号；经济性筛选确保仅对有经济潜力的候选方案付出完整精确评估的代价；两个评估阶段共享同一配电网仿真引擎，仅在时间采样密度上存在差异，从机制上消除了传统代理模型中代理模型与真实模型之间的语义鸿沟。

（2）通过约束分级比较方法，以三级约束逐级比较替代传统罚函数加权，从机制上消除了罚函数系数人为调参的困难。电网安全约束相对于设备可行约束具有绝对的优先排序权，设备可行约束相对于经济可行约束亦然——安全的关键性不被经济指标稀释，不会出现"高收益补偿高风险"的错误排序。不同精度评估结果之间的比较也得到了恰当的隔离处理。

（3）通过优化目标改进率驱动的自适应种群规模调节，使种群规模随搜索进展动态调整：在优化改进显著时自动扩大种群以增强探索，在改进停滞时自动收缩种群以加强局部精化并减少计算浪费。调节信号直接来源于优化目标本身（NPV 改进率），而非依赖外部的种群多样性指标。
